# MEP Projects Analytics

This repository contains SQL queries and analyses for MEP projects over the past decade. The dataset includes company, project, client, and financial metrics used to analyze market trends, revenue, costs, and operational efficiency.

## Contents

- `queries.sql` - Main SQL scripts for market share, client analysis, financial & operational metrics, time trends, and joint ventures.
- `summary_steps.docx` - Word document with step-by-step insights derived from the analysis.
- `README.md` - This file.

## Setup

1. Ensure you have access to a SQL Server instance.
2. Load your sanitized dataset into the `MEP_Projects_Analytics` database.
3. Open the `queries.sql` file in your SQL client (e.g., SSMS, Azure Data Studio) and execute the queries in order.

> ⚠️ **Note:** These SQL scripts do not contain any sensitive credentials. Replace any placeholder database connections with your own.

## Analysis Steps

The analyses are organized in steps:

1. Company Market Share Trends
2. Project Type Trends
3. Client, Subsector, and Industry Analysis
4. Regional and City Revenue Analysis
5. Financial and Operational Metrics
6. Customer Concentration
7. Time-based Trends (Annual and Quarterly)
8. Competition & Joint Ventures

Refer to `summary_steps.docx` for detailed insights and observations.
